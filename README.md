# DogTrips - Frontend

## Descripción
DogTrips es una aplicación web responsiva construida en React para la gestión integral de servicios de paseos de perros. Ofrece una interfaz moderna e intuitiva para que los dueños de mascotas puedan explorar paquetes, reservar paseos y administrar la información de sus mascotas.

## Tecnologías Utilizadas
- **React**: Biblioteca principal para la construcción de interfaces de usuario.
- **React Router**: Para la gestión de rutas y navegación en la aplicación (Single Page Application).
- **Vanilla CSS**: Para estilos consistentes y responsivos sin dependencias externas pesadas.
- **Vite**: Herramienta de construcción rápida y empaquetador moderno para desarrollo de frontend.

## Estructura del Proyecto
El código fuente principal reside en la carpeta `src/`, la cual incluye:
- `components/`: Componentes reutilizables de la interfaz gráfica (e.g. Footer, Navbar).
- `views/`: Las distintas páginas o vistas que componen la aplicación (e.g. Home, Login, Services).

## Instrucciones de Instalación
Sigue estos pasos para ejecutar el proyecto en un entorno local:

1. **Requisitos Previos**: Asegúrate de tener instalado [Node.js](https://nodejs.org/) (recomendado v16 o superior).
2. **Clonar/Navegar al repositorio**: Ubícate en la carpeta `proyectoPrimerParcial`.
3. **Instalar Dependencias**:
   ```bash
   npm install
   ```
4. **Ejecutar en Desarrollo**:
   ```bash
   npm run dev
   ```
   Esto iniciará un servidor local (típicamente en `http://localhost:5173`) con Hot Module Replacement (HMR).
